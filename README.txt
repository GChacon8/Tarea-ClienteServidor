En este repositorio ya se encuentran los .class compilados, en otro caso se podrian borrar y ejecutar "javac *.java" en el folder para generarlos. Para el proyecto se debe ejecutar el servidor y una vez iniciado se debe ejecutar el Cliente, la clase Mensaje solo servira de intermediaria para la conversion de datos.

Cabe destacar que el proyecto no es exactamente lo solicitado, pero fue lo logrado despues de horas de investigacion y entendimiento de java y sockets. Se puede notar una buena comunicacion entre el cliente y el servidor por medio de sockets y el calculo de los datos, por lo que el objetivo de funcionalidad se logro al menos en su mayoria.
